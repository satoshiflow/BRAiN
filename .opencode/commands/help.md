# OpenCode Help

Available slash commands:

- `/help`: show this help overview
- `/loop <mission scope>`: run a mission in continuous execution loop mode

Mission loop resources:

- command definition: `.opencode/commands/loop.md`
- execution policy: `.opencode/policies/execution_loop_policy.md`
- runtime state: `.opencode/state/current_loop.md`

Examples:

- `/loop Implement auth refactor`
- `/loop Continue roadmap implementation`
- `/loop Audit repository architecture`
- `/loop Fix failing tests`
