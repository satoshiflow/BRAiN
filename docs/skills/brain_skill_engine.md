# BRAiN Skill Engine

Die Skill Engine ist die Runtime-Schicht von BRAiN.

## Komponenten

Skill Selector  
Skill Registry  
Skill Planner  
Capability Resolver  
Execution Engine  
Evaluator  
Optimizer

## Pipeline

Intent
→ Skill Selector
→ Skill Planner
→ Capability Resolver
→ Execution Engine
→ Evaluator
→ Memory
→ Optimizer

## Prinzip

Skill Creator erstellt Skills.

Skill Engine führt Skills aus, bewertet sie und verbessert sie.
