# BRAiN Agent Roles for Evolution

## 1. GOTT
Strategische Meta-Instanz.
Verantwortlich für Richtung, Prioritäten, Nutzen, Architekturtreue und langfristige Evolution.

## 2. AXE
Bewusste Eingabe- und Kommunikationsschicht.
Verantwortlich für Aufnahme und Strukturierung von Nutzerwünschen, Signalen und externen Impulsen.

## 3. BRAiN Supervisor
Operative Systemführung.
Verantwortlich für Missionen, Ressourcen, Priorisierung und Laufzeitkoordination.

## 4. Horizon Agent
Technologie- und Trendbeobachtung.
Verantwortlich für Erkennung neuer Entwicklungen in KI, Robotik, Open Source, Tools und Nutzerbedarf.

## 5. Governance Agent
Prüft Regeln, Grenzen, Freigaben und Risiken.

## 6. Immune Agent
Erkennt Schäden, Regressionen, riskante Änderungen und Anomalien.

## 7. Healthcare Agent
Bewertet Systemgesundheit, Wartungsbedarf, technische Hygiene und langfristige Stabilität.

## 8. OpenCode Dev Agent Layer
Setzt Entwicklungs- und Reparaturmissionen um:
- Code
- Module
- Plugins
- Skills
- Workflows
- Refactoring
- Patches
- Upgrades

## Grundregel
Horizon erkennt.
GOTT priorisiert.
Supervisor beauftragt.
Governance prüft.
Immune und Healthcare sichern ab.
OpenCode entwickelt.
