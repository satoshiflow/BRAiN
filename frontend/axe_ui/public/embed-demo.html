Widget API

Once initialized, control the widget via JavaScript:

// Open the widget panel window.AXEWidget.open(); // Close the widget panel window.AXEWidget.close(); // Check if panel is open console.log(window.AXEWidget.isOpen()); // Send a message programmatically window.AXEWidget.sendMessage("Hello, AXE!"); // Access configuration console.log(window.AXEWidget.config);

Development Notes

Local Testing:
  1. Start backend: cd backend && python -m uvicorn main:app --reload
  2. Start frontend: cd frontend/axe_ui && npm run dev
  3. Open this page: http://localhost:3002/embed-demo.html
  4. Open DevTools (F12) → Console to see debug logs

Environment Variables:

  • Backend must allow local requests: AXE_FUSION_ALLOW_LOCAL_REQUESTS=true
  • Frontend embed script base URL: http://localhost:3002

Troubleshooting

Widget not appearing?
  1. Check browser console (F12) for errors
  2. Verify origin is in allowlist (currently: http://127.0.0.1:3002)
  3. Ensure backend is running on http://localhost:8000
  4. Check network tab for failed requests to /api/axe/*
Chat messages not sending?
  1. Backend must have AXE Fusion module enabled
  2. Verify AXE_FUSION_ALLOW_LOCAL_REQUESTS=true in backend env
  3. Check that /api/axe/chat endpoint is accessible
  4. Review backend logs for validation errors
